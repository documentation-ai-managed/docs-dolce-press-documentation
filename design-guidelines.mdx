## Design Preparation

Creating effective designs for printing requires attention to resolution, color modes, and file formats. Follow these guidelines to ensure your printed materials look professional and accurate.

<Callout kind="alert">Always work with high-resolution files to avoid pixelation and poor print quality.</Callout>

<Tabs>
  <Tab title="File Formats" icon="file-text">
    Preferred formats: PDF, AI, PSD. Minimum 300 DPI resolution for all images.
  </Tab>
  <Tab title="Color Modes" icon="palette">
    Use CMYK color mode for printing. RGB is suitable only for digital proofs.
  </Tab>
  <Tab title="Bleed & Safe Zones" icon="crop">
    Include 1/8 inch bleed and keep important elements 1/4 inch from trim edge.
  </Tab>
</Tabs>

<Steps>
  <Step title="Plan Layout" icon="layout">
    Sketch your design and consider how it will look when printed.
  </Step>
  <Step title="Choose Fonts" icon="type">
    Use standard fonts or embed custom fonts in your PDF files.
  </Step>
  <Step title="Prepare Images" icon="image">
    Ensure all images are high-resolution and in the correct color mode.
  </Step>
  <Step title="Add Bleed" icon="scissors">
    Extend backgrounds and images beyond the final trim size.
  </Step>
  <Step title="Convert to PDF" icon="file">
    Export as PDF with embedded fonts and images.
  </Step>
  <Step title="Proof Check" icon="check-circle">
    Review your design for errors before uploading.
  </Step>
</Steps>

<Expandable title="Common Design Mistakes" default-open="false">
  <Expandable title="Low Resolution Images" default-open="false">
    Images below 300 DPI will appear blurry when printed at actual size.
  </Expandable>
  <Expandable title="Missing Bleed" default-open="false">
    Designs without bleed may have white edges after trimming.
  </Expandable>
  <Expandable title="RGB Colors" default-open="false">
    RGB colors may shift when converted to CMYK for printing.
  </Expandable>
</Expandable>

<CodeGroup tabs="Design Checklist,Color Conversion">
  ```javascript
// Design validation function
const validateDesign = (file) => {
  const checks = {
    resolution: file.dpi >= 300,
    colorMode: file.colorMode === 'CMYK',
    hasBleed: file.bleed >= 0.125,
    embeddedFonts: file.fontsEmbedded
  };
  return checks;
};
  ```
  ```python
from PIL import Image

# Convert RGB to CMYK
def rgb_to_cmyk(image_path):
    img = Image.open(image_path).convert('CMYK')
    img.save(image_path.replace('.jpg', '_cmyk.jpg'))
    return 'Converted to CMYK'
  ```
</CodeGroup>
